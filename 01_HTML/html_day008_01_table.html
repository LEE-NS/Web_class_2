table

  • table은 표를 만들 때 사용하는 태그이다.
  • 과거 홈페이지 제작 방식이였다. 수정이 어렵고, 잘 망가져서 약간씩 수정 후 확인을 반복해야 한다.
  • 행(row)과 열(column)로 구성된다. 오(가로) / 열(세로), 행(가로) / 종(세로)
  • table > tr > th/td 의 순서로 마크업한다. tr / td / th 모두 자식 요소로 들어오는 태그이다.
  • table 에 속성값을 추가해서 선을 만들고(border="선 굵기"), 선의 간격(cellspacing="간격 수치")과 셀의 여백 (cellpadding="여백 수치"), 표의 너비(width="너비 값"), 표의 높이(height="높이 값")을 조절할 수 있다.
  • tr의 align 속성으로 내용의 글자를 정렬한다.
  • tr : 행(row)을 정의할 때 사용하는 태그이다.
  • td : 열(column)을 정의할 때 사용하는 태그이다.
  • th : td와 같은 기능인데, 제목 역할을 하는 셀을 정의할 때 사용한다. bold가 기본스타일인듯
