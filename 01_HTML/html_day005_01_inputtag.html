input

input 태그는 type 속성에 따라 다양한 폼을 생성한다.(type 값에 따라 정보의 형태가 달라짐)

placeholder 라는 속성에 값을 입력하면 input 박스 안에 입력 내용에 대한 힌트가 나타난다. 내용이 입력되면 사라진다.

form

form 태그는 입력양식을 만들기 위한 태그다. 입력 양식에 사용하는 태그들은 form 태그에 작성을 해 주면 된다.

아이디 비밀번호

label / required

label : label은 속성이 아니라 태그이다. 입력 태그들에 대한 제목을 정의하는 태그입니다.

required : 항목에 내용이 비었을 경우 경고메시지를 표시하는 속성입니다. 내용이 요구됨.




textarea

textarea 태그는 여러 줄의 글을 입력할 수 있는 텍스트 상자다. textarea 태그는 inline 요소이다.
