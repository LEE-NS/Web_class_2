video / source

  1. 4k 비디오 다운로드 프로그램을 이용
  2. Youtube 등 비디오가 있는 사이트의 링크를 복사(화면 최상단의 링크 주소 등)
  3. 4k 비디오에서 링크 복사 버튼 클릭 > 다운로드(용량과 화질을 고려.)
  4. video 태그 작성
  5. video 태그 안에 source 태그 열기> src=""로 경로 지정
  6. video 태그의 속성
    • controls : 컨트롤 패널을 표시해줌. 재생, 일시정지 등
    • width / height : 영상의 가로/세로 사이즈를 정해주는 속성
    • autoplay : 자동재생
    • muted : 음소거
    • Ex) 켜자말자 자동재생되는 영상만 나오는 경우(정지 안되고 계속 재생) > autoplay와 muted를 함께 사용한 경우이다.
    • loop : 반복재생


iframe

  1. 영상의 html 문서 주소를 넣어서 사용
  3. 동영상을 우클릭하여 소스를 복사하여 옮기면 iframe 태그의 형태로 나타난다.
  4. 간혹 막혀있는 경우도 있음. > 그럴 경우에는 다운로드 해서 쓰던지 ㅇㅇ
