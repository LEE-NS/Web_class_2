position : 요소의 위치 지정밥법 유형을 결정

  1. static : 원래 상태 (default)
  2. relative : absolute 영역의 기준
  3. absolute : 상위 요소의 relative를 기준으로 위치를 재배치
    relatve가 상위요소(형제 태그가 아닌 부모 태그 또는 그 이상)에 위치하고, absolute가 하위요소(형제 태그가 아닌 자식 태그 또는 그 이하)에 위치한다.
    더 가까운 상위요소에 position 속성값을 가진 요소가 있다면, 그 요소를 기준으로 배치된다.
  4. fixed : 요소 위치를 스크린을 기준으로 배치
  5. sticky : 필수적으로 top, bottom, left, right 중에 하나를 설정해 주어야 하고, 설정된 위치에 도달하기 전 까지는 static 속성처럼 행동하다가 설정된 위치에 다다르면 fixed 처럼 행동한다.
  6. 같이 사용되는 스타일 속성 : top or bottom, left or right (각각 상or하/좌or우에서 얼마나 올 것인지)
    단위 : px, %, auto(할당했던 값을 무효화 시킴, 0을 주면 왼쪽 상단으로 오지만 이 상태가 무효화 된 상태)
  7. span 태그나 a 태그 같은 인라인 요소에 absolute 값을 주게 되면 블락 속성을 가지게 된다.
