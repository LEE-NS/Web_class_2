class : 태그에 이름을 줄 때 사용한다. 여러개의 태그에 같은 이름을 사용할 수 있다. 띄워쓰기로 하나의 태그에 여러개의 클래스 이름을 사용할 수 있다.


id : 태그에 이름을 줄 때 사용한다. 여러개의 태그에 같은 이름을 사용할 수 없다.(유일한 이름이 되어야 함) 아이디와 클래스는 이름이 중복되어도 사용할 수 있다.


클래스나 아이디의 값은 영어로 써야한다. 대/소문자 구분을 해야 함. 시작자리에 기호는 큰따옴표("")밖에 들어갈 수 없다. 기호는 영어로 먼저 쓰고 다음에 -, ~, _ 등을 이용하여 추가 작성. 가급적이면 내용과 관련있는 영문명을 붙인다.

head 태그의 title 태그 아래에서 시작한다. 중괄호({})를 사용한다. 클래스는 '.(마침표)', 아이디는 '#'으로 불러온다. 나중에 작성한 값이 우선순위로 적용된다. class보다 id가 우선순위로 적용된다.
