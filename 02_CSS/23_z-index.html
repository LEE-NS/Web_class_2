z-index : 요소의 겹치는 순서를 지정

음수, 양수 모두 사용가능. 숫자가 높을수록 겹쳤을 때 더 상단으로 나타남.

반드시 position과 함께 사용. (absolute, relative 상관없이 같이 사용)

position을 쓰지 않고 단독으로 사용시 적용되지 않는다.

같은 순서값을 가진 요소는 더 나중에 작성된 요소가 더 위에 나타난다.

형제 태그가 아니더라도 position 값을 가진 요소끼리는 z-index 값이 적용됨.

부모요소가 자식요소보다 앞으로 나오는 것은 불가능하다. (값 차이 줘서 억지로 보내려고 하지 말라는 말)

요소의 위치값이 변경되지 않아야 할 경우 position: relative 값을 줘서 z-index와 사용해주면 된다.
