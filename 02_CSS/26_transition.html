transition : 시작에서 종료까지의 변화를 가시화 한다

transition: all 0s;(property duration)

속기법
transition: propertyduration timing-function;

transition은 마우스를 올리고 내릴 때 가시화 되어야 하므로 :hover 와 같이 쓰게 된다. :hover 전에 쓰는지 후에 쓰는지에 따라 효과의 차이가 있다.

마우스를 올리고 내릴 때 모두 가시화 하고 싶다면 :hover 를 사용하기 전 상태에서 transition을 적용시키면 된다.

  1. transition-property: transition을 적용할 css 속성 이름;
  2. transition-duration: transition이 일어나는 지속시간(0s);
  3. transition-timing-function: 속성의 중간값을 계산;
  4. transition-delay: 대기시간, 속성이 변한 시점과 transition이 실제로 시작하는 사이에 기다리는 시간(0s);
