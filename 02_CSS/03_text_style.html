문자 관련 스타일

  • font-size : 글자 크기 조정

    단위 : px, %, em, rem

    em은 부모 요소의 사이즈를 기준으로 곱한다고 생각하면 된다.

    em, rem 더 알아보기

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용페이스를 제공하기 위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 UI/UX 디자인을 검토한다.

  • font-family : 글자체 조정

    font-family: dotum; → 영문 글자체 이름을 쓰는 경우에는 속성값에 영문 글자체명을 적어주면 된다.

    font-family: 돋움; → 한글 글차페 이름을 쓰는 경우에는 속성값에 ''로 묶어서 글자체명을 적어주면 된다.

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용페이스를 제공하기 위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 UI/UX 디자인을 검토한다.

  • font-weight : 글자 굵기 조정

    속성값 : normal, bold, bolder, ligther / 별도의 단위 없이 100부터 100단위로 900까지 사용 가능하다.

    normal 이 4~500 정도, 그 이상은 bold, 이하는 lighter 로 생각하면 된다.

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용페이스를 제공하기 위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 UI/UX 디자인을 검토한다.

  • line-height : 행간 조정

    단위 : px, %, em

    글자 크기가 클수록 더 크게 줘야한다. 글자 크기보다 작으면 글자가 겹침

    통상 작성되는 본문의 행간은 1.4 ~ 1.6em 정도가 적당함.(대체로 그렇다.)

    줄이 한 줄 일때는 행간 값이 하단 높이값이다. 그래서 글자가 한 줄만 있을때는 높이값 없이 line-height 값을 높이 값으로 주면 가로로 가운데 정렬이 이루어진다.

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용페이스를 제공하기 위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 UI/UX 디자인을 검토한다.

  • color : 글자 색상 조정, 배경색은 background-color로 조정

    색상명(red, green, blue 등)은 정확한 영문 색상명이 들어가야 한다. 색상코드(HEX)(#000000, #FFFFFF)도 사용할 수 있다. 색상 코드는 앞에 꼭 #을 붙여야 한다. 색상코드에서 6개의 동일한 숫자 또는 알파벳이 반복되는 경우에는 3개를 생략해서 사용할 수 있다. (#000000 = #000, #FFFFFF = #FFF). RGBA(N, N, N 0 ~ 1사이의 소수)(R 수치, G 수치, B 수치, alpha(opacity))(여기에 사용되는 수치는 0 ~ 255 까지 사용 가능하다.) 도 사용할 수 있다. RGB는 RGBA에서 opacity 수치가 빠진 것.
