CSS 스타일시트 연결하기

  • inline 스타일

    태그 옆(open 태그 안)에 style 속성으로 적어주는 방법

    style은 속성과 속성값이 있다.
    style="속성: 속성값;" 의 형식으로 사용된다

    box1

  • 내부 스타일시트

    CSS를 HTML 문서 내부에 스타일 태그로 적는 방법

    head 태그 안에서 style 태그로 작성한다.

    스타일을 줄 대상의 태그를 작성하고, {속성: 속성값;} 으로 작성한다.

  • 외부 스타일시트

    스타일 속성들을 CSS 파일로 따로 저장해서 HTML 문서에 링크 시키는 방법

    보통의 홈페이지들은 이 방식을 사용하여 제작한다.

    홈페이지가 커지면 일일이 스타일을 줄 수 없다.
    한 개 씩 넣다보면 스크롤이 지나치게 길어져서 이후에 수정하거나 색인이 어려워진다.
    통상은 외부스타일 시트를 CSS 파일을 만들어서 따로 저장해서 HTML 문서에 링크시켜서 사용한다.

    1. Ctrl + N 으로 새 창을 연다.
    2. CSS 확장자명의 파일을 저장해둔다.
    3. 여기에 태그 {속성: 속성값;} 으로 스타일 속성값을 작성한다.
    4. 한가지 씩 뿐만 아니라, 한 번에 여러가지 속성도 넣을 수 있다.
    5. HTML 문서 안에 head 태그에서 link 태그에 href 속성으로 경로를 연결해서 사용한다.
