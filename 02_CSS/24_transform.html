transform : 요소를 여러가지 방법으로 변형

  1. translate : 위치이동
    translateX / translateY
  2. scale : 크기변경
    글자랑 사진도 같이 강제로 크기가 변경되기 때문에 사용 빈도는 낮다
    scaleX / scaleY
  3. rotate : 회전
    rotateX /rotateY
  4. skew : 뒤틀기
    사용빈도 낮음
    skewX / skewY



transform: translate(x,y)

transform: rotate(각도 deg)
transform: skew(각도 deg, 각도 deg)
