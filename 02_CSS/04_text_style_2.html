문자 관련 스타일 2

  • word-spacing : 단어 사이의 간격 조정

    단위 : px, em

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기 위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 UI/UX 디자인을 검토한다.

  • text-decoration : 텍스트 장식

    속성값 : underline, overline, line-through, none

    텍스트에 선 긋기

    링크 태그는 밑줄과 글자색이 기본으로 들어온다.

  • text-indent : 들여쓰기

    단위 : px

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기 위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 UI/UX 디자인을 검토한다.

  • text-shadow : 글자 음영

    text-shadow: (offset-x)(x축 수치)px (offset-y)(y축 수치)px (blur)(블러 수치)px color(색상)

    글자 음영

    blur 수치는 양수만 가능. x, y 수치는 음수가 되면 음영의 위치가 각각 반대로 적용된다.

    color는 hex, rgb 다 먹음

    rgba값으로 음영 주기

  • font-style : 글자의 표현 형식

    속성값 : normal, italic, oblique

    글자 표현 형식 나타내기(oblique)

    글자 표현 형식 나타내기(italic)

    address 시멘틱 태그는, italic이 기본 스타일이다. (이건 속성값을 normal로 줌)
