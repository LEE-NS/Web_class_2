display : block;

인라인요소를 블락으로 변경시켜준다. 또는 보이지 않는 요소를 보이게 할 때 사용한다. 개행이 일어난다.

display: inline;

block을 inline 요소로 변경시켜준다. inline 이므로 개행이 일어나지 않는다.

display: inline-block;

인라인이나 블락을 inline-block으로 변경시켜준다. 인라인을 가지고 있기 때문에 개행이 일어나지 않음. 하지만 width, height, text-align 등 블락에서만 사용할 수 있는 속성을 사용할 수 있다.

inline-block 값을 준 요소의 상위 태그에 text-align 속성을 적용시키면 나열된 요소의 그룹이 정렬된다.
